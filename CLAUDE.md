# HanQuran Project Instructions

## Bahasa

Semua dokumen yang dibuat harus menggunakan Bahasa Indonesia.

Gunakan Bahasa Indonesia untuk:

- Judul
- Subjudul
- Penjelasan
- Diagram
- Tabel
- Keterangan

Pengecualian:

- Nama file
- Nama folder
- Nama route URL
- Nama komponen React
- Interface TypeScript
- Nama library
- Kode program

tetap menggunakan bahasa aslinya.
